# BOOTH帳簿サポーター（Chrome拡張機能） 

このChrome拡張機能は、BOOTHの購入履歴および売上管理画面から、
確定申告や帳簿作成に必要な「仕訳データ（CSV）」と「証憑画像」を効率的に抽出・保存するためのツールです。

---

## 1. インストール方法

1.  Google Chromeで「拡張機能」管理画面（`chrome://extensions/`）を開きます。
2.  右上の「デベロッパーモード」のスイッチをオンにします。
3.  「パッケージ化されていない拡張機能を読み込む」ボタンを押し、本ツールのフォルダ(booth-bookkeeping-supporter)を選択します。※zipは解凍してください。
4.  ツールバーのパズルアイコンから「BOOTH帳簿サポーター」をピン留めすると便利です。

#### 💡 推奨設定（効率化）
Chromeの設定で「**ダウンロード前に各ファイルの保存場所を確認する**」を**オフ**にすると、
画像保存やCSV出力が中断されずスムーズに進められるため推奨しています。

**設定方法**:
1. Chromeの右上メニュー（︙）から「**設定**」を開きます。
2. 左メニューの「**ダウンロード**」を選択します。
3. 「**ダウンロード前に各ファイルの保存場所を確認する**」のスイッチを**オフ**にします。
4. （作業完了後に元の設定に戻すことをお勧めします）

## 2. 主な機能

### ① 購入履歴からの仕訳CSV生成
BOOTHで購入した商品のデータを、各種会計ソフト（freee、弥生会計、マネーフォワードなど）にそのまま取り込める形式で書き出します。

*   **使い方**:
    1.  [BOOTH購入履歴ページ](https://accounts.booth.pm/orders)を開きます。
    2.  本ツールのポップアップを開き、「貸方勘定（決済手段）」を入力します（例：事業主借、普通預金など）。
    3.  出力形式を選択し、必要に応じて抽出期間を指定します。
    4.  「購入履歴CSVを生成」をクリックします。

### ② 証憑画像の一括保存
指定した期間内の注文詳細画面を自動的にスクリーニングし、1ページずつスクリーンショットを保存します。

*   **特徴**: 指定した抽出期間に該当する注文をすべて自動で撮影・保存します。
*   **ファイル名形式**:
    *   **標準**: `booth_order_注文番号.png`
    *   **電帳法対応**: `日付_株式会社ピクシブ_金額_注文詳細_注文番号.png`
*   **注意**: 解析および撮影には少し時間がかかります。実行中はブラウザのタブが自動で開かれます。

### ③ 売上管理からの仕訳CSV生成（ショップ運営者向け）
ショップの月毎の売上、BOOTH手数料、振込入金額、振込手数料を自動集計して仕訳データを作成します。

*   **使い方**:
    1.  [BOOTHショップ管理ページ](https://manage.booth.pm/)を開きます。
    2.  「借方勘定（振込先預金名）」を入力します（例：普通預金、Paypal、事業主貸など）。
    3.  「振込手数料」を選択します（銀行振込 200/300円、または PayPal 0円）。
    4.  「月毎売上CSVを生成」をクリックします。
    *   ※「出力形式」と「抽出期間」は、上の「購入履歴」セクションで設定したものが使用されます。

## 3. 進行状況の確認方法

解析や保存の進捗状況は、ブラウザのデベロッパーツール（**ショートカット F12キー**）の「**コンソール (Console)**」画面ですべて確認することができます。

---

## 4. 対応フォーマット

| 形式名 | 特徴 |
| :--- | :--- |
| **簡易帳簿形式（単式）** | 表計算ソフト（Excel等）での管理や、白色申告・青色申告（10万円控除）に適したシンプルな形式。 |
| **仕訳帳形式（複式）** | 一般的な複式簿記の形式。 |
| **freee用CSV（複式）** | クラウド型会計ソフト「freee」のインポート形式に対応。 |
| **弥生会計用CSV（複式）** | 「弥生会計」「やよいの青色申告」のインポート形式（仕訳日記帳）に対応。 |
| **マネーフォワード用CSV（複式）** | 「マネーフォワード クラウド会計」のインポート形式に対応。 |
